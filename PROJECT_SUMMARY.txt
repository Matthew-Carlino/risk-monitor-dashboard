================================================================================
SETUP & USAGE
================================================================================

INSTALLATION:
  1. Navigate to project: cd /sessions/festive-dreamy-wozniak/repos/risk-monitor-dashboard
  2. Install dependencies: pip install -r requirements.txt
  3. Run dashboard: streamlit run app.py
  4. Open browser to http://localhost:8501

SAMPLE DATA:
  - Loads automatically on first run
  - 15-asset crypto portfolio
  - 90 days of realistic price history
  - No API keys or external data required

CUSTOM DATA:
  1. Prepare CSV: asset, position, entry_price, current_price, notional, exchange
  2. Use sidebar "Upload CSV" button
  3. Dashboard generates synthetic price/funding data

TESTING:
  - Run tests: pytest tests/ -v
  - 18 unit tests covering all core metrics
  - All tests pass with known inputs/outputs
